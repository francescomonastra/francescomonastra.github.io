This page collects codes, analysis tools, and simulation-related software that I develop or contribute to.

## Geant4 detector simulations

Codes and tools related to detector-background simulations, event analysis, and particle-interaction studies.

**HENABICS** — an ongoing project extending particle-background characterization beyond simulation, combining Geant4 mass models with real background and radiation-environment data from operating X-ray missions such as XMM-Newton and eROSITA.

Repositories will be added as they become publicly available.

## Machine learning methods for high-energy astrophysics

**MAGI** (Multivariate Autoencoder for particle Generative Inference) — a generative model that learns to sample physically consistent Geant4 particle events directly, to speed up detector-background simulations. Currently applied to cosmic-ray background modeling for the NewAthena X-IFU instrument. See the [Research](/research/) page for more on the underlying architecture.

Repositories will be added as they become publicly available.
